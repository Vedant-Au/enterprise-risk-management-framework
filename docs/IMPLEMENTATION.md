# ERM implementation roadmap

| Phase | Objective | Exit evidence |
| --- | --- | --- |
| 1. Foundation | Approve policy, appetite, taxonomy and ownership | Board approval and owners for every unit |
| 2. Register | Run workshops, score risks and assign treatments | No risk without an owner; red/amber actions agreed |
| 3. Monitoring | Define KRIs and quarterly group reporting | Monthly KRIs for every red risk; first board report |
| 4. Integration | Embed ERM in investment, contracts and planning | Risk criteria evidenced in board decisions |

The academic recommendation targets repeatable/managed maturity within 24 months. Timing must be resourced and validated by the organisation.
